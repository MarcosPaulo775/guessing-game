## Guessing Game

### How to start
Open the project with [devcontainer in Visual Studio](https://code.visualstudio.com/docs/remote/containers) Code and after initialization run the command below
```
cargo run
```

### Dependencies
- Visual Studio Code
- WSL2 (Only Windows)
- Docker
- Docker Compose
